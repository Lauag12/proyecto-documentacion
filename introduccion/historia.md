#  Historia y Evolución del Desarrollo Web

El desarrollo web ha experimentado una transformación drástica, pasando de páginas estáticas a complejas aplicaciones interactivas. Esta evolución se puede dividir en tres eras clave: la Web Estática, la Web Dinámica y la Web 2.0 (o Interactiva).

## 1. 1991 - 2000: La Web Estática (Web 1.0)

La web nació como una herramienta para compartir documentos. No había distinción clara entre Frontend y Backend, ya que el contenido era fijo.

* **Fundación:** Tim Berners-Lee inventa HTML, HTTP y la primera web en el CERN.
* **Tecnologías:** Principalmente **HTML** para la estructura, con CSS apareciendo al final de la década para dar estilo.
* **Experiencia:** Los sitios eran meros "libros en línea". El usuario solo podía leer la información.
* **Desarrollo:** No se necesitaba una base de datos ni lógica de servidor compleja.

## 2. 2000 - 2005: La Web Dinámica y el Backend (Web 1.5)

El surgimiento de la necesidad de interactuar con datos (inicios de foros, comercio electrónico) impulsó la creación de la lógica del lado del servidor.

* **Aparición del Backend:** Lenguajes como **PHP**, **Perl** y **Java** permitieron generar HTML "sobre la marcha" usando información de **bases de datos**.
* **Separación de Roles:** Se consolida la necesidad de desarrolladores **Backend** (quienes gestionan la lógica y los datos) y **Frontend** (quienes manejan el diseño).
* **Ejemplos:** Sitios que ya permitían iniciar sesión, hacer búsquedas y guardar preferencias.

## 3. 2005 - Actualidad: La Web Interactiva (Web 2.0 y Más Allá)

La revolución de **JavaScript** en el navegador y el auge de las redes sociales redefinieron la experiencia web. Las aplicaciones empezaron a comportarse como programas de escritorio.

* **AJAX:** Permitió que las páginas se actualizaran sin recargarse completamente, lo que llevó a interfaces mucho más rápidas y fluidas.
* **Frameworks Frontend:** Nace la necesidad de gestionar la complejidad del lado del cliente, dando paso a librerías y *frameworks* como **jQuery**, y posteriormente, **React**, **Angular** y **Vue**.
* **Evolución del Backend:** Surgimiento de **Node.js**, permitiendo que JavaScript se ejecute también en el servidor, consolidando el rol del desarrollador **Full-Stack** que usa el mismo lenguaje en ambas capas.

---

##  El Futuro del Desarrollo Web

Hoy en día, el desarrollo se enfoca en el rendimiento, la accesibilidad y la experiencia móvil, utilizando tecnologías como WebAssembly, arquitecturas *serverless* y la inteligencia artificial para crear experiencias más inmersivas y personalizadas.
